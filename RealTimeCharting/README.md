RealTimeCharting
================

This is a real-time charting demo with TeeChart for Xamarin.Android. 

This application configures the chart and refreshes the data to optimize the performance and responsiveness of the application.

## Sample Requirements

You'll need TeeChart for Xamarin.Android evaluation or registered version to run this example. Fully functional evaluation version can be obtained at http://www.steema.com/download/mobile
